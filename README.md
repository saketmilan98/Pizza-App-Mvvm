# Pizza-App-Mvvm
This app has ui for ordering customisable pizzas. 
When ordering, the user can customise the crust & size of the pizzas. The user can order as many pizzas as they wish.

Technology Stack :

Kotlin

MVVM architecture

Live Data

ViewModel

Coroutines

Databinding

Glide

Apk file is present in project root.
